# 涛哥的交易系统

> 核心原则：主力是方向，估值是安全带，止损是保险绳。三根绳少一根都是裸奔，三根全系好才敢上路。
>
> 量价升级版：融入威科夫量价体系，让信号更精准！

---

## 第一维：识主力——搞清楚谁在操盘

### Step 1：看股东结构（定性质）

| 查什么 | 怎么查 | 判定 |
|:---|:---|:---:|
| 十大流通股东 | 东方财富→个股→股东研究 | 基金/社保/外资扎堆=机构票 |
| 基金持仓家数 | 同上 | 0家=游资票，>10家=机构票 |
| 股东人数变化 | 同上，看季报环比 | 减少=筹码集中✅ 增加=散户接盘🚩 |
| 港股通增减 | 东方财富→沪深港通 | 减仓=外资不看好🚩 |

**黄金法则：基金0持仓 + 股东人数暴增 = 高危游资票，别碰！**

### Step 2：看资金流向（定方向）

| 信号 | 判定 | 操作 |
|:---|:---|:---:|
| 股价涨 + 主力净流入 | ✅ 主力真实进场 | 持有/跟进 |
| 股价涨 + 主力净流出 | 🚩 诱多出货 | 减仓/远离 |
| 股价跌 + 主力净流入 | ✅ 主力暗中吸筹 | 关注/择机进 |
| 股价跌 + 主力净流出 | 💀 主力真跑路 | 赶紧跑 |

**查法：看5日/10日趋势，别只看1天！**

### Step 3：看走势特征（定阶段）

| 阶段 | 特征 | 操作信号 |
|:---|:---|:---:|
| 建仓期 | 横盘震荡，小阴小阳，跌不动 | 🟡 观察，别急 |
| 拉升期 | 量价齐升，突破前高 | 🟢 跟进最佳窗口 |
| 洗盘期 | 快速下跌但缩量，很快拉回 | 🟡 拿住别慌 |
| 出货期 | 放量滞涨或放量下跌 | 🔴 跑！别幻想 |

**洗盘 vs 出货 区分：**

| 特征 | 洗盘 | 出货 |
|:---|:---|:---|
| 成交量 | 缩量（缩小到拉升期1/3以下） | 放量（甚至天量） |
| 回调幅度 | 一般不超过10% | 跌破关键支撑位 |
| 回调速度 | 快跌快拉（1-3天） | 阴跌不止（连续多日） |
| 资金流向 | 主力小幅流出或净流入 | 主力大幅净流出 |

## 第二维：判估值——搞清楚贵不贵

### Step 1：三大估值指标必查

| 指标 | 查法 | 安全区 | 危险区 |
|:---|:---|:---|:---|
| PE（市盈率） | 东方财富→个股→估值分析 | <行业均值 | >行业均值2倍以上🚩 |
| PB（市净率） | 同上 | <3倍 | >8倍🚩 |
| PEG | PE÷利润增速 | <1（低估） | >2（高估）🚩 |

### Step 2：同行业对比（最关键！）

必须和同板块标的横向对比PE/PEG/基金持仓，单独看一个数字没有意义。

**金安国纪教训：** PEG看起来0.10很便宜？但基金0持仓=专业投资者不认！PE 73倍+0基金=便宜没人要，一定有原因！

### Step 3：估值红线——触到就走

以下任一条件触发，不管主力在不在，坚决不进：

1. PE > 行业均值 2倍以上
2. 股价年内涨幅 > 150%，但业绩增速跟不上
3. 基金0持仓 + 高PE = 双杀信号
4. 融资余额占流通市值 > 5%（杠杆过高）

## 第三维：止损保命——搞清楚错了怎么办

### Step 1：入场前设好三条线

| 止损线 | 设法 | 作用 |
|:---|:---|:---|
| 硬止损 | 买入价 ×（1-8%），到点无脑砍 | 保命底线，防深套 |
| 技术止损 | 前期低点/重要均线（MA10/MA20）下方1-2% | 防破位加速下跌 |
| 时间止损 | 买入后3-5天不涨反跌，出局 | 防耗时间成本 |

### Step 2：持仓中的移动止损（锁利润）

| 浮盈阶段 | 止损线位置 | 目的 |
|:---|:---|:---|
| 5%以内 | 不动（硬止损保护成本） | 保本 |
| 5%-15% | 上移到成本价 | 保证不亏 |
| 15%-30% | 移到+5% | 保证赚钱走 |
| 30%以上 | 跟5日均线走 | 让利润奔跑 |

**口诀：止损只往上移，绝不往下挪！**

### Step 3：仓位管理——别一把梭

| 信号强度 | 仓位 | 操作 |
|:---|:---:|:---|
| 三维全绿（主力在+估值低+趋势好） | 3成底仓 | 先试水 |
| 持仓盈利+趋势确认 | 加到5-7成 | 顺势加仓 |
| 任何一维亮红灯 | 减到3成以下 | 防御收缩 |
| 两维以上亮红灯 | 清仓走人 | 别犹豫 |

## 第四维：量价实战 —— 威科夫量价体系

### 威科夫三大定律

1. **供需定律**：放量突破是真突破，缩量到前高是假突破，放量破位是真破位
2. **因果定律**：没有足够的底部横盘（因），就没有足够的大行情（果）。趋势高度 ≈ 底部宽度 ÷ 3
3. **量价一致律**：放量大涨=健康；放量不涨=见顶；放量小跌=可能见底；缩量上涨=不牢

### 九大量价信号

1. **Spring（弹簧效应/破底翻）** → 🟢 买入：价格短暂跌破前低→迅速收回→放量
2. **UT（Upthrust/冲高回落）** → 🔴 卖出：价格短暂冲过前高→迅速回落→放量
3. **Stopping Volume（天量止跌）** → 🟢 买入：下跌后天量K线→价格并未继续大跌
4. **SC（Selling Climax/恐慌抛售）** → 🟢 关注：大幅下跌+天量成交，等二次测试确认
5. **BC（Buying Climax/抢购高潮）** → 🔴 卖出：大幅上涨+天量成交
6. **Absorption（吸筹/主力暗渡陈仓）** → 🟢 关注：低位震荡，频繁上下影线+成交量不规则放大
7. **Test（测试/主力试盘）** → 缩量Test=合格✅ 放量Test=还有抛压❌
8. **LPS（Last Point of Support/最后支撑点）** → 🟢 最佳买入点！缩量回踩支撑→放量启动
9. **DE（Demand Exhaustion/需求衰竭）** → 🔴 卖出：上涨趋势末端→价涨量缩/价平量增

### 威科夫主力操盘五阶段

- **阶段A — 吸筹建仓期**：SC→AR→ST/Spring，横盘3-6月，振幅收窄，大盘跌它不跌，仓位10-20%
- **阶段B — 洗盘震仓期**：Spring/Shakeout，3天洗盘：大阴破位放量→缩量续跌→放量阳线收回，仓位20-40%
- **阶段C — 首次拉升期**：脱离底部，沿20日线缓步攀升，阶梯式放量，仓位50-70%
- **阶段D — 主升浪**：加速上涨，远离均线，可能缩量涨停，仓位50-60%
- **阶段E — 顶部派发/出货期**：BC→UTAD→SOW，天量暴涨→冲高回落，仓位0-20%

### 三周期共振分析法

```
看周线：定方向、定阶段 → 确定上升/下降趋势，处于A-E哪一阶段
看日线：找信号、找买卖点 → 寻找Spring/LPS/Stopping Volume等
看60分：定精确入场、定止损 → 精细化买入位置
```

**三重确认→买入 | 两重→观望 | 一重或无→放弃**

## 量价止损法

| 止损类型 | 规则 |
|:---|:---|
| **放量破支撑** | 跌破关键价位+放量 = 立即出 |
| **缩量回踩不破** | 缩量到支撑位未破 = 持有 |
| **天量止盈** | 放历史天量 = 减半或清仓 |
| **连续缩量止盈** | 连续3日缩量上涨 = 减仓 |
| **Spring止损** | Spring最低价下方2-3% |

## 五大戒律

1. 不追高 —— 任何买入都等缩量回调
2. 不死扛 —— 放量破支撑 = 走人
3. 不抄底 —— 等缩量企稳 + 放量反弹再进
4. 不贪婪 —— 达到目标位分批减仓
5. 不恐慌 —— 缩量回调是机会不是风险
